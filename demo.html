How to Use WellNest

The following outlines the features used in the video:

  • 🌟 Add, edit, delegate, and delete tasks.
  • 📅 Access daily tasks via the read-aloud feature.
  • 📝 Add, edit, and delete appointments.
  • 🔔 Retrieve scheduled appointments with the voice read-aloud feature.
  • ✅ Edit task status, whether completed or ongoing.
  • 💊 Add, edit, and delete prescriptions.
